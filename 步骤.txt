1.要建一个git的仓库

   git init //初始化一个仓库

   git add .   //添加到暂缓区

   git commit -m "初始化项目"  //提交

   git remote add origin https://github.com/lixinmusic/LearnMiniProgram.git //关联远程仓库

   git push -u origin master //把代码上传到github上

   

   git tag 01_知识点01

   git log //查看有哪些提交

   git reset 2c3cc72cf  //回退到了2c3cc72cf这个版本

   git reset --hard 2c3cc72cf  //强制回退到2c3cc72cf这个版本

   git tag //这个可以查看上传tag的个数

   git push --tags //这个是push到远程仓库
